A simple tetris game
====================

Controls:
---------

Left/Right arrows: Move back and forth

Up arrow: Rotate piece

Down arrow: Drop piece one space

A/S keys: Rotate piece left/right

Space bar: Full drop

추가 예정인 기능 :
---------

1. 음악 및 효과음 추가
2. 멈춤기능 추가
3. 메인화면 추가
4. 게임오버 되어도 다시 시작 할 수 있게 만들기
